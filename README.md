# e-Plant Shopping Application

## Project Description
The e-Plant Shopping application is a web-based project that allows users to browse, select, and purchase plants online. The application provides a simple and user-friendly interface for viewing plant products, managing a shopping cart, and placing orders.

## Features
- Browse available plants
- View plant details
- Add plants to cart
- Simple and responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
